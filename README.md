# Clef - Frontend

Web user interface for Clef, an educational platform for teachers and students focused on syllabus and teaching flow management.

**Stack:** React 19, Vite 8, Tailwind CSS 4, React Router 7, TanStack Query 5, Axios

## Getting Started

1. Clone the repo and navigate to the frontend directory
2. Copy `.env.example` to `.env` and fill in your values (e.g., your backend API URL)
3. Install dependencies by running `npm install`
4. Start the development server by running `npm run dev`

## Requirements

- Node.js (v18 or newer)
- npm (Node Package Manager)
